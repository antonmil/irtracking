Installation
============
This section describes how to get cemtracking running under Linux.
Open a terminal window.

Get the code

    hg clone https://bitbucket.org/amilan/irtracking    
    
Start MATLAB and run `compileMex.m` to build the utilities binaries.


Running
=======

goto ./contracking
run `runAll.m`
